# react-game
### New Game
---
Для начала новой игры нажать New Game. Доступно 2 режима игры: Игрок против Игрока и Игрок против Компьютера. Режим меняется нажатием радиобаттона.

### Continue
---
Есть возможность сохранить процесс игры. Для этого необходимо нажать кнопку Save Game на игровом поле. Для продолжение игры нажать Continue в главном меню.

### Settings
---
Для настройки игры нажать Settings в меню. Можно Вкл/Выкл звук и музыку, регулировать громкость звуков и музыки, менять цвет поля и фигур. Также есть настройка время на ход. Установив определенное время, каждому игроку дается на ход установленное время, по истечении которого игрок проигрывает автоматически.

### Statistics
---
Для просмотра статистики игры нажать Statistics в меню. Здесь отображается таблица с исходами всех игр в двух режимах. Для сброса статистики нажать Reset statistics.
